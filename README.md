# Automatas
Author: Fabian Hernando Vera Carrillo
Convert NFDA to DFA on python 

### Features
- Define qintuple for DFA
- Define qintuple for NFDA
- Convertion NFA to DFA
- Run automata's
- Minimization of automata (in progress)
